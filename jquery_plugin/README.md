# LinkIt jQuery Plugin

**LinkIt** is a simple jQuery plugin to attach links to elements.

Version 0.1.0

## Usage

```
<script>
  $(document).ready(function(){
    $('span').linkIt({
      href: 'http://google.com',
      text: 'Click Me',
      target: '_blank'
    });
  });
</script>
```
